## 6. HOW TO USE IN SYSTEM PROMPTS

### Recommended format for the AI tutor:
When the AI explains a topic, it can say:
> "For more detail, see OpenStax Chemistry 2e, Chapter 7 — free at openstax.org/books/chemistry-2e/pages/7-introduction"

Or for interactive exploration:
> "Try the PhET Molecule Shapes simulation to visualise this — free at phet.colorado.edu/en/simulations/molecule-shapes"

### Embedding in system prompts:
Add a reference block at the end of each subject's system prompt:

```
FREE RESOURCES — direct students to these when they want to read further:
- OpenStax Chemistry 2e: openstax.org/books/chemistry-2e (free textbook, covers all topics)
- LibreTexts Chemistry: chem.libretexts.org (detailed notes on every topic)
- PhET Simulations: phet.colorado.edu (interactive visualisations)
- Khan Academy: khanacademy.org/science/chemistry (video explanations)
When referencing, say: "For more detail, see [resource name] — it's free at [URL]."
```

**For the AGF Companion (commercial product at £29/month):**
- CC BY content (OpenStax, PhET): Can be used and adapted with attribution
- CC BY-NC-SA content (LibreTexts, Khan, MIT): Can link to but NOT embed in a paid product without permission. Linking is always fine.
- Safest approach: **Link to resources** rather than embedding content. This is what the system prompt references do.
